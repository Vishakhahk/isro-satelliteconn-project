About ISRO and This Project

The Indian Space Research Organisation (ISRO) uses satellites for communication, weather, agriculture, disaster management, and national development. This project demonstrates the power of space applications using open data and web technologies.

Built using HTML, CSS, JavaScript, Leaflet, and public APIs. Inspired by ISRO's mission to serve the nation.
